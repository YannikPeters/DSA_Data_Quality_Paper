# DSA_Data_Quality_Paper
A Git Repository containing parts of the data and code of the paper "Little Mentioning, Moderate Attention, Great Relevance –  The Quality of Online Platform Data in the Digital Services Act" in Platforms & Society. When using data and code, the multiple correspondence analysis of the paper is reproducible. 

Peters, Yannik, and Katrin Weller. 2026. "Little mentioning, moderate attention, great relevance: The quality of online platform data in the Digital Services Act." Platforms & Society 3: 1-17. doi: https://doi.org/10.1177/29768624261438624.
